## üìò Project Overview

This project presents an in-depth Exploratory Data Analysis (EDA) of a used-car dataset from CarLoop, an Indian automotive e-commerce platform.

The goal of this project is to uncover meaningful insights related to pricing, vehicle condition, brand popularity, fuel preferences, seller behavior, and market trends.

Unlike predictive modeling workflows, this project is entirely exploratory and focuses on:

- Understanding relationships between variables  
- Detecting patterns and anomalies  
- Discovering factors that influence used-car pricing  
- Summarizing business-ready insights  

---

## üéØ Problem Statement

CarLoop aims to strengthen its used-car marketplace by optimizing pricing, improving customer experience, and understanding buyer preferences.

To achieve this, the platform needs deeper insights into its historical car listings.

As a data analyst, your task is to examine the dataset, analyze trends, and derive insights that help answer key business questions such as:

---

## üíº Key Business Questions

### Pricing & Depreciation
- What factors most strongly influence the selling price of used cars?  
- How do age, mileage, and engine capacity impact depreciation?

### Inventory & Demand
- Which brands and models are most frequently listed?  
- What fuel types and transmissions are most preferred by customers?

### Vehicle Condition
- How does mileage vary across brands or car segments?  
- Are there patterns in listing behavior based on region or seller type?

### Operational Insights
- What seller types dominate the market‚Äîdealers or individuals?

## üìÇ Data Dictionary

Below is a simplified, concise description of all columns included in the dataset:

| Column Name | Description |
|-------------|-------------|
| **Make** | Brand/manufacturer of the vehicle. |
| **Model** | Specific model name of the car. |
| **Price** | Selling price of the used car in INR. |
| **Year** | Manufacturing year of the car. |
| **Kilometer** | Total kilometers driven by the vehicle. |
| **Fuel Type** | Type of fuel the car uses. |
| **Transmission** | Type of transmission (manual/automatic). |
| **Location** | City or region where the car is listed. |
| **Color** | Exterior color of the car. |
| **Owner** | Ownership status (1st owner, 2nd owner, etc.). |
| **Seller Type** | Type of seller (individual/dealer/certified). |
| **Engine** | Engine displacement measured in CC. |
| **Max Power** | Maximum power produced by the engine. |
| **Max Torque** | Maximum torque produced by the engine. |
| **Drivetrain** | Drivetrain type (FWD/RWD/AWD). |
| **Length** | Length of the vehicle in mm. |
| **Width** | Width of the vehicle in mm. |
| **Height** | Height of the vehicle in mm. |
| **Seating Capacity** | Number of seats in the vehicle. |
| **Fuel Tank Capacity** | Fuel tank capacity in liters. |
